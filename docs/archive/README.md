# Архив проектных документов

Файлы в этом каталоге являются замороженными историческими материалами.

Они не считаются активным планом работ, не определяют приоритеты текущих заданий и не обновляются автоматически. Выполнение или изменение архивного roadmap допускается только после прямой команды пользователя с указанием архивного документа или пункта.
